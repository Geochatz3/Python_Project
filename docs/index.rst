Cursor Python Project documentation
===================================

This documentation covers the structure, usage, and development practices for the
Cursor Python Project. The application demonstrates generating random ``Circle`` models,
formatting analytical statistics, and exposing a CLI that orchestrates the workflow.

Use the navigation below to learn how to install dependencies, run the CLI,
execute the automated test suite, and explore the API internals.
