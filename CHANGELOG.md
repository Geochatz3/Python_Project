# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-01-XX

### Added
- Structured logging with JSON output format option
- Configuration file support (TOML)
- Environment variable configuration
- JSON output format for programmatic use
- Comprehensive test suite
- Sphinx documentation
- GitHub Actions CI/CD pipeline
- Code quality tools (ruff, black)

### Changed
- Restructured project into `src`-layout package
- Enhanced CLI with multiple configuration sources
- Improved error handling and validation

## [0.1.0] - 2025-01-XX

### Added
- Initial release
- Basic circle generation and statistics
- Command-line interface
- Text-based output
